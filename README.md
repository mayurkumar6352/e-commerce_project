# E-commerce Sales Analysis & Customer Segmentation

## Project Overview
This project analyzes e-commerce sales data to understand customer behavior, sales trends, and business performance.

## Problem Statement
The business wanted to identify high-value customers, inactive customers, sales trends, and optimize marketing budget allocation.

## Tools Used
- Python (Pandas, NumPy)
- Excel
- Power BI

## Key Steps
- Cleaned raw transactional data
- Created revenue and KPI metrics
- Analyzed monthly sales trends
- Performed RFM customer segmentation
- Built decile analysis
- Designed Power BI dashboard

## Key Insights
- A small group of customers contributes most of the revenue
- Sales peak during specific months
- Champions and Loyal customers should receive higher marketing focus

## Deliverables
- Jupyter Notebook
- Power BI Dashboard
- Project Presentation

## What I Learned
- Data cleaning and preparation
- Customer segmentation using RFM
- Translating data into business insights

## Business Impact
- Identified high-value customer segments to improve retention.
- Highlighted at-risk customers for re-engagement.
- Provided data-backed guidance for marketing budget allocation.
- Enabled quick decision-making through a single interactive dashboard.
